# Zoe Phiri Portfolio

A refined portfolio and enquiry site for Zoe Phiri, a South African web and brand designer. The site presents services, selected project examples, pricing tiers, and a Netlify-powered contact form for project enquiries.

## Key Technologies

- TanStack Start with React 19 and file-based routing
- Vite for local development and production builds
- Tailwind CSS 4 plus custom global CSS for the visual system
- Lucide React for interface icons
- Netlify Forms for contact form handling on deploy

## Local Development

Install dependencies, then run the development server:

```bash
npm install
npm run dev
```

The Netlify deployment uses the build command and publish directory defined in `netlify.toml`.
